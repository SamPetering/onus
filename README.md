# onus 
##### it's a to-do list

### Dev
* yarn
* yarn electron-dev

### Download Installer
[.dmg download](http://www.mediafire.com/file/fkqjappi7477f60/onus-0.1.0.dmg/file)

### Package locally
* git clone https://github.com/SamPetering/onus
* cd onus
* yarn
* yarn build
* yarn electron-pack
* installers will be available in the dist directory
