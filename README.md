# onus 
##### it's a to-do list

### Dev
* ```yarn```
* ```yarn electron-dev```

### Package locally
* ```git clone https://github.com/SamPetering/onus```
* ```cd onus```
* ```yarn```
* ```yarn build```
* mac:```yarn electron-pack``` windows: ```yarn electron-pack-windows```
* installers will be available in the dist directory
